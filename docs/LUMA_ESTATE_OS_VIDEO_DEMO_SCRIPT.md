# Luma Estate OS - Video Demo Script

## Instrucciones de Grabación
- Grabación de pantalla combinada con cámara (burbuja o lateral) para mantener conexión humana.
- **Qué mostrar primero:** Empezar mostrando el caso segmentado (Vista del Río), luego el backend/dashboard, no al revés.
- **Qué NO decir:** Evitar tecnicismos (React, Next.js, API, bases de datos). Hablar siempre de "infraestructura, terminales de captación, atribución, pipeline".
