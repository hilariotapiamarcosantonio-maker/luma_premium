# Luma Estate OS - Guía Operativa de Grabación

## 1. Preparación antes de grabar

**Configuración del Entorno Visual:**
- **Resolución:** Grabar pantalla en 1920x1080 (Full HD) o formato 4K si tu equipo lo soporta.
- **Navegador:** Usar un perfil de navegador limpio (Chrome/Arc/Safari) sin barras de marcadores personales, ni extensiones visibles (ej. adblockers).
- **Zoom:** Recomendado entre 90% y 100% para que el diseño respire bien sin verse gigante.
- **Pestañas (Abrir solo estas URLs y en este orden):**
  1. `https://luma-premium.vercel.app/luma-estate-os`
  2. `https://luma-premium.vercel.app/luma-estate-os/vista-del-rio`
  3. `https://vista-del-rio-next.vercel.app/inversor` *(Si tienes estas rutas funcionales)*
  4. `https://vista-del-rio-next.vercel.app/corporativo`
  5. `https://vista-del-rio-next.vercel.app/retiro`
  6. `https://luma-premium.vercel.app/luma-estate-os/diagnostico`
- **Audio:** Asegura un micrófono de buena calidad, sin eco. Entorno silencioso.
- **Guion:** Tenerlo impreso o en una segunda pantalla/tablet para leer de corrido sin bajar la mirada.

## 5. Qué NO Decir (Reglas de Oro)
- NO decir "es una hoja de Google Sheets" (Hablar de "backend ágil o base de datos estructurada").
- NO mencionar librerías o plataformas (Next.js, Vercel, código, servidores).
- NO decir "te hago una página web" o "te diseño una web bonita".
- NO mencionar precios en el video (esto debe manejarse en la llamada de diagnóstico).
- NO sonar rogado ni desesperado. El tono es de un consultor experto explicando una máquina.
- NO prometer "inteligencia artificial" o "automatizaciones locas de WhatsApp" si no están en Foundation. Prometer **Visibilidad y Organización**.
